# 整行用法

## 整行剪下（Cut line）

- Windows / Linux：`Ctrl + X`（沒選取也會直接剪整行）
- macOS：`⌘ + X`

## 整行複製（Copy line）

- Windows / Linux：`Ctrl + C`
- macOS：`⌘ + C`

## 向上／下複製一行

- Windows / Linux：`Ctrl + D`
- macOS：`⌘ + D`

## 整行刪除（Delete line）

- Windows / Linux：`Ctrl + Y`
- macOS：`⌘ + ⌫（Command + Delete）`

## 快速跳到「行尾」

- Windows / Linux：`End`
- macOS：`⌘ + →`

## 快速跳到「行首」

- Windows / Linux：`Home`
- macOS：`⌘ + ←`

### 沒有 Home / End

- Windows
  👉 `Fn + →`（通常等於 End）
  👉 `Fn + ←`（通常等於 Home）

## 選取到「行尾」

- Windows: `Shift + End`
- macOS: `⌘ + Shift + →`

## 「整行操作」高頻鍵

| 功能         | Windows           | macOS                |
| ------------ | ----------------- | -------------------- |
| 複製整行     | `Ctrl + D`        | `⌘ + D`              |
| 上移一行     | `Shift + Alt + ↑` | `Shift + Option + ↑` |
| 下移一行     | `Shift + Alt + ↓` | `Shift + Option + ↓` |
| 向下新增一行 | `Shift + Enter`   | `Shift + Enter`      |
