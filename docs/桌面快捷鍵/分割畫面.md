# 分割畫面

Windows 快速分割畫面，最常用是這幾組：

## 1. 左右分割

先選到要操作的視窗，按：

- `Win + ←`：貼到左半邊
- `Win + →`：貼到右半邊

很適合一邊看文件、一邊寫 code。

---

## 2. 四分割

先把視窗貼到左右，再往上或往下：

- `Win + ←` 再按 `Win + ↑`：左上
- `Win + ←` 再按 `Win + ↓`：左下
- `Win + →` 再按 `Win + ↑`：右上
- `Win + →` 再按 `Win + ↓`：右下

---

## 3. 最大化 / 還原 / 最小化

- `Win + ↑`：最大化
- `Win + ↓`：還原；再按一次通常會最小化

---

## 4. Windows 11 更方便的方法

如果你是 **Windows 11**，可以用 **Snap Layouts**：

### 方法 A：滑鼠

把滑鼠移到視窗右上角的 **最大化按鈕** 上停一下，會出現分割版型，例如：

- 左右各半
- 三欄
- 四格

然後直接選位置。

### 方法 B：快捷鍵

- `Win + Z`

會直接叫出可選的分割版型。

---

## 5. 超實用搭配

### 切換視窗

- `Alt + Tab`

### 多桌面切換

- `Win + Ctrl + ← / →`

### 顯示桌面

- `Win + D`
