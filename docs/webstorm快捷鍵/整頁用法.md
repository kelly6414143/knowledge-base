# 整頁用法

## 🔽 直接跳到「檔案最底部」

### 🖥️ Windows / Linux

👉 **`Ctrl + End`**

### 🍎 macOS

👉 **`⌘ + ↓`**

---

## 🔼 順便給你「跳到最上面」

### 🖥️ Windows / Linux

👉 **`Ctrl + Home`**

### 🍎 macOS

👉 **`⌘ + ↑`**

---

## 💡 進階補充（真的好用）

如果你只是想：

- 👀 快速捲動到底 → 滾輪 + `Ctrl`
- 🚀 跳到某一行 → `Ctrl + G`（mac 是 `⌘ + L`）
- 🧭 快速搜尋檔案內 function → `Ctrl + F12`（檔案結構）
