# 時間搓

## 最快做法：先用公式顯示，再貼上為值

假設你的試算表本身是 **台北 `+08:00`**，你想取 **`+07:00`** 現在時間。

先在某格輸入：

```excel
=NOW()-TIME(1,0,0)
```

再把該格：

1. 複製 **Ctrl + C**
2. 貼上特殊格式 → **Ctrl + Shift + V**

這樣就會把當下時間「凍結」下來，不會再跳動。

若想直接顯示成完整格式，可用：

```excel
=TEXT(NOW()-TIME(1,0,0),"yyyy-mm-dd hh:mm:ss")&" +07:00"
```

這種適合：

- 臨時記錄時間點
- 手動複製到紀錄表
- 不需要每次都重新算

---

## 直接改整份試算表時區

如果你大多數時間都在記錄 **`+07:00`**，最省事的是直接把該試算表時區改成 `GMT+07:00`。

位置：
**檔案 → 設定 → 時區**

改完後：

- `NOW()`
- `TODAY()`
- `Ctrl + Shift + ;`

都會直接以 `+07:00` 為準。

這樣你要記錄時，只要：

- 用 `Ctrl + ;`
- 或 `Ctrl + Shift + ;`

就能直接插入固定當下時間，不用再手動 `-1 小時`。
