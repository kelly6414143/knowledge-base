# 向量資料庫

## chromaDB

## 安裝 python

```
python --version
```

> 如果出現類似：Python 3.10.11 或 Python 3.9.x ➜ 代表已安裝 ✅  
> 如果顯示「不是內部或外部命令」之類 ➜ 代表尚未安裝，需要先去 python.org 下載安裝（安裝時記得勾選 Add Python to PATH）

## 建立 Python 虛擬環境（這個只給 Chroma 用，避免影響其他專案）

```
python -m venv .venv
```

## 啟動 Python 虛擬環境

- Command Prompt（cmd）：

  ```
  .venv\Scripts\activate
  ```

- PowerShell：
  ```
  .venv\Scripts\Activate.ps1
  ```

### 啟動成功

命令列前面會多一個像這樣的東西

```
(.venv) D:\chroma-server>
```

## 在虛擬環境安裝 chromadb

在`顯示 (.venv)` 的視窗執行

1. 先更新 pip  
   `python -m pip install --upgrade pip`
2. 安裝 chromadb  
   `pip install chromadb`

## 啟動 Chroma Server

在`顯示 (.venv)` 的視窗執行

```
chroma run --host 0.0.0.0 --port 8000
```
