# 時間搓

## 最快做法：先用公式顯示，再貼上為值

假設你的試算表本身是 **台北 `+08:00`**，你想取 **`+07:00`** 現在時間。

先在某格輸入：

```excel
=NOW()-TIME(1,0,0)
```

再把該格：

1. 複製 **Ctrl + C**
2. 貼上特殊格式 → **Ctrl + Shift + V**

這樣就會把當下時間「凍結」下來，不會再跳動。

若想直接顯示成完整格式，可用：

```excel
=TEXT(NOW()-TIME(1,0,0),"yyyy-mm-dd hh:mm:ss")&" +07:00"
```

這種適合：

- 臨時記錄時間點
- 手動複製到紀錄表
- 不需要每次都重新算

---

## 直接改整份試算表時區

如果你大多數時間都在記錄 **`+07:00`**，最省事的是直接把該試算表時區改成 `GMT+07:00`。

位置：
**檔案 → 設定 → 時區**

改完後：

- `NOW()`
- `TODAY()`
- `Ctrl + Shift + ;`

都會直接以 `+07:00` 為準。

這樣你要記錄時，只要：

- 用 `Ctrl + ;`
- 或 `Ctrl + Shift + ;`

就能直接插入固定當下時間，不用再手動 `-1 小時`。

## Excel 增加函式

### 一、適合的做法

你可以建立一個命名函式，例如：

- `NOW_TZ7`
- `CURRENT_TIME_UTC7`
- `GET_TIME_GMT7`

然後讓它內部公式固定回傳 **+07:00** 的目前時間。

---

### 二、建立方式

在 Google Sheet：

**資料 → 命名函式**

接著新增一個命名函式。

### 範例 1：直接回傳日期時間文字

函式名稱可設為：

```text
NOW_TZ7
```

公式填：

```excel
=TEXT(NOW()-TIME(1,0,0),"yyyy-mm-dd hh:mm:ss")&" +07:00"
```

> 這是假設你的試算表時區是 **台灣 +08:00**，所以減 1 小時變成 +07:00。

之後你在任一儲存格輸入：

```excel
=NOW_TZ7()
```

就會得到像這樣的結果：

```text
2026-03-12 10:25:30 +07:00
```

### 三、命名方式

如果你主要是自己用，建議簡短好記：

- `NOW_TZ7`
- `NOW_TZ8`
- `NOW_TZ9`

如果你是要給團隊一起用，建議清楚一點：

- `CURRENT_DATETIME_UTC7`
- `CURRENT_DATETIME_UTC8`
